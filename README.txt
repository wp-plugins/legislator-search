== Description ==

This plugin adds an AJAX-enabled widget that allows users to quickly and easily search for congressional legislators
by zip code.

This plugin requires a Sunlight Labs API key.

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip and upload `legislator_search` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
